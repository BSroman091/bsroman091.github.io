1. ¿Qué es CSS y para qué sirve?

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje utilizado para definir la apariencia visual de una página web. Mientras que HTML se encarga de la estructura (títulos, párrafos, imágenes), CSS se utiliza para darle estilo, es decir, colores, tamaños, espacios, alineaciones y diseño en general.

Gracias a CSS, una página web puede pasar de ser un documento simple y sin formato a una interfaz atractiva y fácil de usar. Por ejemplo, puedes cambiar el color de fondo, modificar el tipo de letra, centrar contenido o incluso crear animaciones.

CSS es fundamental en el desarrollo web moderno, ya que permite mejorar la experiencia del usuario y hacer que los sitios sean visualmente agradables y profesionales.

Ejemplo Básico

<!DOCTYPE html>
<html>
<head>
<style>
body {
  background-color: lightblue;
}
h1 {
  color: navy;
  text-align: center;
}
</style>
</head>
<body>

<h1>Hola mundo</h1>

</body>
</html>

En este ejemplo, CSS cambia el fondo a azul claro y centra el texto.

2. Separación entre Contenido y Diseño

Una de las principales ventajas de CSS es que permite separar el contenido (HTML) del diseño (CSS). Esto significa que puedes tener un archivo HTML limpio con la información, y otro archivo CSS que controle cómo se ve esa información.

Esta separación hace que el código sea más fácil de leer, mantener y modificar. Por ejemplo, si quieres cambiar el color de todos los títulos, solo necesitas modificar una línea en el archivo CSS en lugar de cambiar cada elemento en HTML.

Además, facilita el trabajo en equipo: un programador puede encargarse del contenido y otro del diseño sin interferir.

Ejemplo Práctico

HTML (index.html):

<h1 class="titulo">Bienvenido</h1>

CSS (estilos.css):

.titulo {
  color: green;
  font-size: 30px;
}

Ventaja: Si deseas cambiar el color, solo editas el archivo CSS. El HTML permanece intacto.

Concepto Clave

La separación de contenido y diseño permite que múltiples desarrolladores trabajen en paralelo: unos en HTML (estructura) y otros en CSS (estilo), sin conflictos.

3. Selectores en CSS

Los selectores son una parte fundamental de CSS, ya que permiten seleccionar los elementos HTML a los que se les aplicará un estilo. Existen varios tipos de selectores, como los de etiqueta, clase, identificador, y combinaciones más avanzadas.

El selector indica "qué elemento quieres modificar", y luego dentro de llaves {} defines los estilos.

Tipos Comunes de Selectores

Selector de Etiqueta

Aplica estilos a todas las instancias de una etiqueta HTML específica.

p {
  color: blue;
}

Selector de Clase

Aplica estilos a elementos que tienen una clase específica. Se identifica con un punto (.).

.miClase {
  font-size: 18px;
}

Selector de ID

Aplica estilos a un elemento único. Se identifica con una almohadilla (#).

#miId {
  background-color: yellow;
}

Ejemplo Completo

CSS:

p {
  color: blue;
}

.miClase {
  font-size: 18px;
}

#miId {
  background-color: yellow;
}

HTML:

<p>Texto azul</p>
<p class="miClase">Texto con tamaño grande</p>
<p id="miId">Texto con fondo amarillo</p>

Selectores Avanzados

  • Selector descendiente: div p → selecciona todos los <p> dentro de <div>
  • Selector de atributo: [type="text"] → selecciona elementos con atributo específico
  • Pseudo-clases: a:hover → selecciona un elemento en un estado específico

4. Propiedades y Valores

En CSS, cada estilo se define mediante una propiedad y un valor. La propiedad indica qué aspecto quieres cambiar (color, tamaño, margen, etc.), y el valor define cómo se verá ese cambio.

La estructura básica es:

propiedad: valor;

Cada línea termina con punto y coma, y todas las propiedades se agrupan dentro de llaves {}.

Ejemplo de Propiedades

h1 {
  color: red;
  font-size: 40px;
  text-align: center;
  margin: 20px;
  padding: 10px;
}

Propiedades Comunes

Color y Fondo

  • color → color del texto
  • background-color → color de fondo
  • background-image → imagen de fondo

Texto

  • font-size → tamaño de la fuente
  • font-family → tipo de fuente
  • text-align → alineación del texto (left, center, right)
  • font-weight → grosor del texto (bold, normal)

Espacio

  • margin → espaciado externo
  • padding → espaciado interno

Dimensiones

  • width → ancho
  • height → altura

Bordes

  • border → borde completo
  • border-radius → esquinas redondeadas
Dato Importante

CSS tiene cientos de propiedades. Estas son solo las más comunes. A medida que practiques, irás aprendiendo más.

5. Cascada y Prioridad en CSS

CSS significa "hojas de estilo en cascada" porque sigue un sistema de prioridad para decidir qué estilos aplicar cuando hay conflictos.

Cuando varios estilos afectan a un mismo elemento, CSS utiliza tres reglas principales:

  • La especificidad → qué selector es más específico
  • El orden → el último estilo escrito tiene prioridad
  • La importancia → uso de !important

Ejemplo 1: Orden de Cascada

p {
  color: blue;
}

p {
  color: red;
}

<p>El texto será rojo</p>

El texto será rojo porque la segunda regla tiene prioridad.

Ejemplo 2: Especificidad

p {
  color: blue;
}

#texto {
  color: green;
}

<p id="texto">El texto será verde</p>

El color será verde porque el ID (#texto) tiene mayor especificidad que la etiqueta (p).

Jerarquía de Especificidad

Selector Especificidad Ejemplo
Etiqueta Baja p { }
Clase Media .miClase { }
ID Alta #miId { }
!important Crítica color: red !important;

Uso de !important

La palabra clave !important fuerza un estilo a aplicarse, independientemente de la cascada. Sin embargo, debe usarse con cuidado.

p {
  color: blue !important;
}

Consejo: Evita usar !important en exceso, ya que hace el código más difícil de mantener.

6. Diseño Responsivo (Responsive Design)

El diseño responsivo es una técnica que permite que una página web se adapte a diferentes tamaños de pantalla, como celulares, tablets y computadoras. Hoy en día es fundamental, ya que la mayoría de las personas navegan desde dispositivos móviles.

CSS hace posible esto mediante el uso de media queries, que permiten aplicar estilos dependiendo del tamaño de la pantalla. Así, puedes cambiar el diseño, tamaño de texto o posición de elementos según el dispositivo.

Esto mejora la experiencia del usuario, ya que evita que tenga que hacer zoom o desplazarse de forma incómoda.

Ejemplo 1: Cambio de Color de Fondo

body {
  background-color: white;
}

@media (max-width: 600px) {
  body {
    background-color: lightgray;
  }
}

En este caso:

  • En pantallas grandes el fondo será blanco
  • En pantallas pequeñas (como celulares) será gris claro

Ejemplo 2: Adaptación de Tamaño de Fuente

h1 {
  font-size: 40px;
}

@media (max-width: 600px) {
  h1 {
    font-size: 25px;
  }
}

El tamaño del título se reduce en pantallas pequeñas para mejorar la legibilidad.

Puntos de Quiebre Comunes

Dispositivo Ancho de Pantalla Media Query
Celular Hasta 480px @media (max-width: 480px)
Tablet 481px - 768px @media (max-width: 768px)
Desktop 769px+ Estilos por defecto

Ejemplo Completo de Responsive Design

/* Estilos para pantallas grandes (Desktop) */
.contenedor {
  display: flex;
  gap: 20px;
}

.sidebar {
  width: 30%;
}

.main-content {
  width: 70%;
}

/* Adaptación para tablets */
@media (max-width: 768px) {
  .contenedor {
    gap: 15px;
  }

  .sidebar {
    width: 25%;
  }

  .main-content {
    width: 75%;
  }
}

/* Adaptación para celulares */
@media (max-width: 480px) {
  .contenedor {
    display: block;
  }

  .sidebar,
  .main-content {
    width: 100%;
    margin-bottom: 20px;
  }
}

7. Tipos de CSS (Inline, Interno y Externo)

CSS se puede aplicar de tres formas diferentes en una página web:

1. CSS Inline

Se aplica directamente dentro de una etiqueta HTML usando el atributo style.

<p style="color: red;">Texto rojo</p>

Ventajas:

  • Rápido para pruebas rápidas

Desventajas:

  • Desordenado y difícil de mantener
  • No se puede reutilizar
  • Mezcla HTML con CSS

2. CSS Interno

Se escribe dentro de la etiqueta <style> en el <head> del documento HTML.

<head>
<style>
p {
  color: blue;
}
</style>
</head>

Ventajas:

  • Útil para páginas pequeñas
  • Mantiene HTML y CSS separados

Desventajas:

  • No se puede reutilizar en otras páginas
  • Aumenta el tamaño del archivo HTML

3. CSS Externo (Recomendado)

Se escribe en un archivo separado .css y se conecta al HTML mediante la etiqueta <link>.

<head>
  <link rel="stylesheet" href="estilos.css">
</head>

Ventajas:

  • Código más limpio y organizado
  • Fácil mantenimiento
  • Reutilizable en múltiples páginas
  • El navegador cachea el archivo, mejorando el rendimiento

Desventajas:

  • Una solicitud HTTP adicional al servidor
Mejor Práctica

Siempre usa CSS Externo para proyectos profesionales. Es la opción más mantenible, escalable y eficiente.

Comparación de los Tres Métodos

Tipo Ubicación Reutilizable Recomendado
Inline En la etiqueta HTML No Solo para pruebas
Interno En <style> No Páginas simples
Externo Archivo .css separado Sí ✅ Siempre

8. Clases e Identificadores (Class vs ID)

En CSS, las clases y los IDs se utilizan para aplicar estilos a elementos específicos. Aunque pueden parecer similares, tienen diferencias importantes.

Clases (class)

  • Se pueden usar varias veces en la misma página
  • Se identifican con un punto (.) en CSS
  • Son reutilizables

Ejemplo:

/* CSS */
.boton {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
}

/* HTML */
<button class="boton">Aceptar</button>
<button class="boton">Enviar</button>
<button class="boton">Cancelar</button>

IDs (id)

  • Solo se usan una vez por página
  • Se identifican con una almohadilla (#) en CSS
  • Son únicos

Ejemplo:

/* CSS */
#titulo {
  color: red;
  font-size: 32px;
}

#encabezado {
  background-color: navy;
  padding: 20px;
}

/* HTML */
<h1 id="titulo">Mi título</h1>
<div id="encabezado">...</div>

Comparación: Class vs ID

Característica Class ID
Símbolo en CSS . (punto) # (almohadilla)
Frecuencia de uso Múltiples veces Una sola vez
Especificidad Media Alta
Caso de uso Estilos reutilizables Elementos únicos
Ejemplo .boton #navegacion

Diferencia Clave

Regla de Oro

class → reutilizable (múltiples elementos con el mismo estilo)
id → único (un solo elemento especial)

Ejemplo Práctico Combinado

/* CSS */
.tarjeta {
  border: 1px solid gray;
  padding: 15px;
  margin: 10px;
}

#contenido-principal {
  width: 70%;
  float: left;
}

.destacado {
  background-color: yellow;
}

/* HTML */
<div id="contenido-principal">
  <div class="tarjeta">Tarjeta 1</div>
  <div class="tarjeta destacado">Tarjeta 2 (destacada)</div>
  <div class="tarjeta">Tarjeta 3</div>
</div>

9. Modelo de Caja (Box Model)

El modelo de caja es uno de los conceptos más importantes de CSS. Todos los elementos HTML se representan como una caja rectangular compuesta por cuatro capas:

Las Cuatro Capas del Box Model

  • Contenido (Content) → lo que se muestra (texto, imagen, etc.)
  • Relleno (Padding) → espacio interno entre el contenido y el borde
  • Borde (Border) → línea alrededor del elemento
  • Margen (Margin) → espacio externo alrededor del borde

Visualización del Box Model

┌─────────────────────────────┐
│ MARGIN │
│ ┌─────────────────────────┐│
│ │ BORDER ││
│ │ ┌───────────────────┐ ││
│ │ │ PADDING │ ││
│ │ │ ┌─────────────┐ │ ││
│ │ │ │ CONTENT │ │ ││
│ │ │ └─────────────┘ │ ││
│ │ └───────────────────┘ ││
│ └─────────────────────────┘│
└─────────────────────────────┘

Ejemplo Práctico

div {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 30px;
}

Esto significa:

  • El contenido mide 200px de ancho
  • Tiene espacio interno (padding) de 20px en todos los lados
  • Un borde sólido negro de 5px
  • Y separación externa (margin) de 30px en todos los lados

Propiedades del Box Model

Width y Height

div {
  width: 300px;
  height: 200px;
}

Padding (Relleno Interno)

/* Todos los lados */
div {
  padding: 20px;
}

/* Lados específicos */
div {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 10px;
  padding-left: 20px;
}

/* Sintaxis abreviada */
div {
  padding: 10px 20px; /* arriba/abajo, izquierda/derecha */
}

Border (Borde)

/* Borde completo */
div {
  border: 2px solid blue;
}

/* Bordes específicos */
div {
  border-top: 1px solid red;
  border-right: 1px solid green;
  border-bottom: 1px solid blue;
  border-left: 1px solid yellow;
}

/* Esquinas redondeadas */
div {
  border-radius: 10px;
}

Margin (Margen Externo)

/* Todos los lados */
div {
  margin: 20px;
}

/* Lados específicos */
div {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 10px;
  margin-left: 20px;
}

/* Centrar horizontalmente */
div {
  margin: 0 auto;
}

box-sizing

De forma predeterminada, el ancho y alto no incluyen padding ni border. Para cambiar esto, usa box-sizing:

/* Por defecto */
div {
  box-sizing: content-box; /* width = contenido */
}

/* Recomendado */
div {
  box-sizing: border-box; /* width = contenido + padding + border */
}
Consejo Profesional

Añade box-sizing: border-box; a todos los elementos al inicio de tu CSS. Esto hace que sea mucho más fácil calcular dimensiones:

* {
  box-sizing: border-box;
}

Ejemplo Completo

.tarjeta {
  width: 300px;
  padding: 15px;
  border: 1px solid #ddd;
  margin: 20px auto;
  border-radius: 8px;
  box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

.tarjeta h2 {
  margin-top: 0;
  padding-bottom: 10px;
  border-bottom: 2px solid #667eea;
}

10. Animaciones y Efectos en CSS

CSS permite crear animaciones y efectos visuales sin necesidad de JavaScript. Esto hace que las páginas sean más dinámicas e interactivas, mejorando la experiencia del usuario.

Existen dos formas principales de crear animaciones en CSS:

1. Transiciones (Transitions)

Permiten cambiar suavemente un valor cuando ocurre un evento (como pasar el mouse o hacer clic).

Sintaxis Básica

elemento {
  transition: propiedad duración velocidad retraso;
}

Ejemplo: Cambio de Color al Pasar el Mouse

button {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  transition: background-color 0.5s ease;
}

button:hover {
  background-color: red;
}

Cuando el usuario pasa el mouse sobre el botón, el color cambia suavemente de azul a rojo en 0.5 segundos.

Propiedades de Transición

  • transition-property → qué propiedad animar (ej. color, width, opacity)
  • transition-duration → duración en segundos (ej. 0.5s)
  • transition-timing-function → velocidad (ease, linear, ease-in, ease-out)
  • transition-delay → retraso antes de comenzar (ej. 0.2s)

Ejemplo Completo

.caja {
  width: 100px;
  height: 100px;
  background-color: blue;
  transition-property: width, background-color;
  transition-duration: 0.3s, 0.5s;
  transition-timing-function: ease-in-out;
}

.caja:hover {
  width: 200px;
  background-color: green;
}

2. Animaciones con @keyframes

Permiten crear animaciones más complejas y personalizadas que se ejecutan automáticamente.

Sintaxis Básica

@keyframes nombreAnimacion {
  from {
    /* estilos iniciales */
  }
  to {
    /* estilos finales */
  }
}

elemento {
  animation: nombreAnimacion duracion iteraciones;
}

Ejemplo: Movimiento Horizontal

@keyframes mover {
  from {
    transform: translateX(0px);
  }
  to {
    transform: translateX(100px);
  }
}

div {
  animation: mover 2s infinite;
}

Esto mueve un elemento de izquierda a derecha continuamente, cada 2 segundos.

Ejemplo: Desvanecimiento (Fade In/Out)

@keyframes desvanecer {
  0% {
    opacity: 0;
  }
  50% {
    opacity: 1;
  }
  100% {
    opacity: 0;
  }
}

.elemento {
  animation: desvanecer 3s infinite;
}

Propiedades de Animación

  • animation-name → nombre de la animación
  • animation-duration → duración en segundos
  • animation-timing-function → velocidad (ease, linear, etc.)
  • animation-delay → retraso inicial
  • animation-iteration-count → cuántas veces se repite (ej. infinite, 3)
  • animation-direction → dirección (normal, reverse, alternate)

Ejemplo Completo: Animación Compleja

@keyframes rotarYCrecer {
  0% {
    transform: rotate(0deg) scale(1);
    opacity: 0;
  }
  50% {
    opacity: 1;
  }
  100% {
    transform: rotate(360deg) scale(1.5);
    opacity: 0;
  }
}

.bola {
  width: 50px;
  height: 50px;
  background-color: #667eea;
  border-radius: 50%;
  animation: rotarYCrecer 2s ease-in-out infinite;
}

Diferencias: Transiciones vs Animaciones

Aspecto Transiciones Animaciones
Inicio Necesita un evento (hover, clic) Se ejecuta automáticamente
Complejidad Simples (de un estado a otro) Complejas (múltiples estados)
Control Menos control Más control con @keyframes
Uso de código transition @keyframes + animation
Ejemplo Botón que cambia color al pasar mouse Carga infinita

Transformaciones (Transform)

Las transformaciones son propiedades CSS que permiten modificar elementos antes de ser renderizados:

/* Trasladar */
transform: translateX(50px);
transform: translateY(100px);

/* Rotar */
transform: rotate(45deg);

/* Escalar */
transform: scale(2);
transform: scaleX(1.5);

/* Sesgar */
transform: skew(10deg);

/* Combinaciones */
transform: translateX(50px) rotate(45deg) scale(1.5);
Rendimiento

Las transiciones y animaciones de CSS son más eficientes que hacer cambios con JavaScript, ya que se ejecutan en el hilo de composición del navegador.
