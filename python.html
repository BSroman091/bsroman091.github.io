1. ¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, lo que significa que puede utilizarse para desarrollar una gran variedad de aplicaciones. Fue creado por Guido van Rossum a finales de los años 80 y lanzado oficialmente en 1991. Desde entonces, ha crecido hasta convertirse en uno de los lenguajes más utilizados en el mundo.

Una de las principales características de Python es su enfoque en la legibilidad del código. Esto significa que su sintaxis está diseñada para ser clara y fácil de entender, incluso para personas que están comenzando a programar. A diferencia de otros lenguajes que pueden ser complejos y difíciles de leer, Python utiliza palabras simples y estructuras ordenadas.

Python es ampliamente utilizado en diferentes áreas como desarrollo web, automatización de tareas, inteligencia artificial, análisis de datos, desarrollo de videojuegos y más. Su versatilidad lo convierte en una herramienta muy poderosa tanto para principiantes como para profesionales.

Además, Python es un lenguaje interpretado, lo que significa que no necesita ser compilado antes de ejecutarse. Esto permite probar código de forma rápida y facilita el aprendizaje.

El Programa Más Simple: Hola Mundo

print("Hola mundo")

Este es el programa más básico en Python, y muestra un mensaje en pantalla. Aunque es simple, demuestra la facilidad de Python para realizar tareas básicas.

Características Principales de Python

  • Legible: Código claro y fácil de entender
  • Interpretado: Se ejecuta sin compilación previa
  • Dinámico: No requiere declaración de tipos
  • Versatil: Múltiples aplicaciones
  • Amplia comunidad: Muchos recursos disponibles
Dato Histórico

El nombre "Python" no viene de la serpiente, sino de la serie de comedia británica "Monty Python". Guido van Rossum era fan de la serie cuando creó el lenguaje.

2. Sintaxis Simple y Legible

Una de las mayores ventajas de Python es su sintaxis sencilla y clara. A diferencia de otros lenguajes como C o Java, Python no utiliza llaves {} para definir bloques de código, sino que usa la indentación (espacios al inicio de cada línea).

Esto obliga a los programadores a escribir código ordenado, lo cual facilita la lectura y comprensión. La indentación no es opcional en Python, es obligatoria, y un error en ella puede hacer que el programa no funcione correctamente.

Este enfoque hace que Python sea ideal para principiantes, ya que permite enfocarse más en la lógica del programa que en la complejidad de la sintaxis.

Ejemplo: Condicional con Indentación

if 10 > 5:
    print("Diez es mayor que cinco")

Aquí, el print está indentado, lo que indica que pertenece al bloque del if. Si no hubiera indentación correcta, el código generaría un error.

Comparación con Otros Lenguajes

Lenguaje Ejemplo Complejidad
Python if x > 5:
print(x)		 Baja
Java if (x > 5) {
System.out.println(x);
}		 Media
C if (x > 5) {
printf("%d", x);
}		 Alta

La Importancia de la Indentación

# Código correcto
if edad > 18:
    print("Mayor de edad")
    print("Puede votar")

# Código incorrecto (error)
if edad > 18:
print("Mayor de edad")  # Error: falta indentación
Regla de Oro

En Python, la indentación es sintaxis. Generalmente se usan 4 espacios (o una tabulación) para cada nivel de indentación.

3. Variables y Tipos de Datos

En Python, las variables se utilizan para almacenar información. A diferencia de otros lenguajes, no es necesario declarar el tipo de variable, ya que Python lo asigna automáticamente dependiendo del valor.

Esto se conoce como tipado dinámico, y permite escribir código más rápido y flexible.

Tipos de Datos Comunes

Enteros (int): Números sin decimales 
edad = 25
numero = -50
cantidad = 1000
Flotantes (float): Números con decimales 
altura = 1.75
precio = 19.99
pi = 3.14159
Cadenas (str): Texto 
nombre = "Luis"
ciudad = 'Madrid'
mensaje = "Hola, ¿cómo estás?"
Booleanos (bool): Verdadero o Falso 
es_estudiante = True
activo = False
mayor_de_edad = True

Ejemplo Completo

nombre = "Luis"
edad = 16
altura = 1.75
es_estudiante = True

print(nombre)
print(edad)
print(altura)
print(es_estudiante)

Cambio Dinámico de Tipos

En Python, puedes cambiar el tipo de una variable en cualquier momento:

valor = 10              # int
print(type(valor))     # <class 'int'>

valor = "texto"        # str
print(type(valor))     # <class 'str'>

valor = 3.14           # float
print(type(valor))     # <class 'float'>

Operaciones Básicas

# Operaciones matemáticas
a = 10
b = 3

suma = a + b          # 13
resta = a - b         # 7
multiplicacion = a * b # 30
division = a / b      # 3.333...
potencia = a ** b     # 1000
modulo = a % b        # 1
Concepto Importante

La función type() te permite verificar el tipo de dato de cualquier variable.

4. Estructuras de Control

Las estructuras de control permiten que un programa tome decisiones y repita acciones. Son fundamentales para crear programas interactivos y funcionales.

Condicionales (if, elif, else)

Permiten ejecutar código dependiendo de una condición.

Ejemplo Básico

edad = 17

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")

Ejemplo con elif

calificacion = 8

if calificacion >= 9:
    print("Sobresaliente")
elif calificacion >= 7:
    print("Notable")
elif calificacion >= 5:
    print("Aprobado")
else:
    print("Suspenso")

Operadores de Comparación

Operador Significado Ejemplo
== Igual a x == 5
!= No igual a x != 5
> Mayor que x > 5
< Menor que x < 5
>= Mayor o igual x >= 5
<= Menor o igual x <= 5

Bucles (for y while)

Permiten repetir un bloque de código varias veces.

Bucle for

for i in range(5):
    print("Número:", i)

# Salida:
# Número: 0
# Número: 1
# Número: 2
# Número: 3
# Número: 4

Bucle while

contador = 0

while contador < 5:
    print("Contador:", contador)
    contador = contador + 1

# Se ejecuta mientras contador sea menor que 5

Ejemplo Práctico

numeros = [1, 2, 3, 4, 5]

for numero in numeros:
    if numero % 2 == 0:
        print(numero, "es par")
    else:
        print(numero, "es impar")

Operadores Lógicos

Operador Significado Ejemplo
and Y lógico (ambas condiciones deben ser verdaderas) x > 5 and y < 10
or O lógico (al menos una condición debe ser verdadera) x > 5 or y < 10
not Negación not x > 5

5. Funciones

Las funciones son bloques de código que se pueden reutilizar para realizar una tarea específica. Esto ayuda a evitar repetir código y facilita la organización del programa.

En Python, las funciones se definen con la palabra clave def.

Función Básica

def saludar():
    print("¡Hola!")

saludar()  # Llamada a la función

Función con Parámetros

def sumar(a, b):
    return a + b

resultado = sumar(5, 3)
print(resultado)  # 8

Función con Valor por Defecto

def saludar(nombre="usuario"):
    print("Hola", nombre)

saludar()              # Hola usuario
saludar("María")       # Hola María

Función con Múltiples Parámetros

def presentacion(nombre, edad, ciudad):
    print("Me llamo", nombre)
    print("Tengo", edad, "años")
    print("Soy de", ciudad)

presentacion("Luis", 25, "Madrid")

Función que Devuelve Múltiples Valores

def obtener_datos():
    nombre = "Ana"
    edad = 20
    return nombre, edad

name, age = obtener_datos()
print(name, age)

Docstrings (Documentación)

def calcular_area_rectangulo(largo, ancho):
    """
    Calcula el área de un rectángulo.
    
    Parámetros:
    - largo: el largo del rectángulo
    - ancho: el ancho del rectángulo
    
    Devuelve:
    - El área del rectángulo
    """
    return largo * ancho

area = calcular_area_rectangulo(10, 5)
print(area)
Ventajas de las Funciones
  • Evitan repetición de código
  • Hacen el código más organizado
  • Facilitan el mantenimiento
  • Permiten reutilizar código

6. Listas y Colecciones de Datos

Python ofrece varias formas de almacenar múltiples datos en una sola variable. Las listas son las más comunes, pero también existen otras estructuras como tuplas, diccionarios y conjuntos.

Listas

Las listas son colecciones ordenadas y modificables.

Ejemplo Básico

frutas = ["manzana", "pera", "uva"]

for fruta in frutas:
    print(fruta)

Operaciones con Listas

numeros = [1, 2, 3, 4, 5]

# Acceder a elementos
print(numeros[0])        # 1
print(numeros[-1])       # 5 (último elemento)

# Agregar elementos
numeros.append(6)        # [1, 2, 3, 4, 5, 6]
numeros.insert(0, 0)     # [0, 1, 2, 3, 4, 5, 6]

# Eliminar elementos
numeros.remove(3)        # Elimina el valor 3
ultimo = numeros.pop()   # Elimina el último

# Longitud
print(len(numeros))      # Cantidad de elementos

Diccionarios

Los diccionarios permiten almacenar datos en pares clave-valor:

persona = {
    "nombre": "Ana",
    "edad": 20,
    "ciudad": "Barcelona"
}

print(persona["nombre"])    # Ana
print(persona["edad"])      # 20

# Agregar elementos
persona["profesion"] = "Ingeniera"

# Recorrer diccionario
for clave, valor in persona.items():
    print(clave, ":", valor)

Tuplas

Las tuplas son similares a listas, pero inmutables (no se pueden modificar):

coordenadas = (10, 20, 30)

print(coordenadas[0])   # 10
print(len(coordenadas)) # 3

# Las tuplas no se pueden modificar
# coordenadas[0] = 15   # Error

Conjuntos (Sets)

Los conjuntos almacenan elementos únicos sin orden:

numeros = {1, 2, 3, 4, 5}
numeros.add(6)          # Agregar
numeros.remove(3)       # Eliminar

# Operaciones de conjunto
a = {1, 2, 3}
b = {3, 4, 5}

union = a.union(b)      # {1, 2, 3, 4, 5}
interseccion = a.intersection(b)  # {3}

Comparación de Colecciones

Tipo Ordenada Modificable Elementos Únicos Ejemplo
Lista Sí Sí No [1, 2, 3]
Tupla Sí No No (1, 2, 3)
Diccionario Sí Sí Claves únicas {"a": 1}
Conjunto No Sí Sí {1, 2, 3}

7. Programación Orientada a Objetos (POO)

Python soporta la programación orientada a objetos (POO), que permite crear estructuras más organizadas y reutilizables.

Se basa en clases y objetos, donde una clase es como un molde y un objeto es una instancia de esa clase.

Concepto Básico

class Animal:
    def __init__(self, nombre):
        self.nombre = nombre

    def hablar(self):
        print("Hace un sonido")

perro = Animal("Firulais")
print(perro.nombre)  # Firulais
perro.hablar()       # Hace un sonido

Atributos y Métodos

Atributos: características del objeto
Métodos: acciones que puede realizar el objeto

class Persona:
    def __init__(self, nombre, edad):
        # Atributos
        self.nombre = nombre
        self.edad = edad
    
    # Métodos
    def presentarse(self):
        print("Me llamo", self.nombre)
        print("Tengo", self.edad, "años")
    
    def cumplir_años(self):
        self.edad = self.edad + 1
        print(self.nombre, "ahora tiene", self.edad, "años")

# Crear objeto
juan = Persona("Juan", 25)
juan.presentarse()
juan.cumplir_años()

Herencia

La herencia permite que una clase herede características de otra:

class Animal:
    def __init__(self, nombre):
        self.nombre = nombre
    
    def hacer_sonido(self):
        print("Sonido genérico")

class Perro(Animal):
    def hacer_sonido(self):
        print("¡Guau!")

class Gato(Animal):
    def hacer_sonido(self):
        print("¡Miau!")

perro = Perro("Rex")
gato = Gato("Misi")

perro.hacer_sonido()  # ¡Guau!
gato.hacer_sonido()   # ¡Miau!

Concepto Importante: self

class Coche:
    def __init__(self, marca):
        self.marca = marca  # 'self' hace referencia al objeto
    
    def mostrar_marca(self):
        print("Este coche es de marca:", self.marca)

mi_coche = Coche("Toyota")
mi_coche.mostrar_marca()  # Este coche es de marca: Toyota
¿Por qué POO?

La POO permite modelar situaciones del mundo real dentro de un programa. Por ejemplo, puedes crear una clase "Estudiante" con atributos como nombre, edad y calificación, y métodos como estudiar() o presentar_examen().

8. Librerías y Módulos

Una de las mayores ventajas de Python es su gran cantidad de librerías. Estas son colecciones de código ya hecho que puedes usar para ahorrar tiempo y esfuerzo.

Existen librerías para prácticamente cualquier tarea: matemáticas, ciencia de datos, inteligencia artificial, desarrollo web, y mucho más.

Librerías Estándar (integradas)

math - Operaciones Matemáticas

import math

print(math.pi)           # 3.141592653589793
print(math.sqrt(16))     # 4.0
print(math.pow(2, 3))    # 8.0
print(math.ceil(4.3))    # 5 (redondea hacia arriba)
print(math.floor(4.7))   # 4 (redondea hacia abajo)

random - Números Aleatorios

import random

numero = random.randint(1, 10)  # Número aleatorio entre 1 y 10
print(numero)

lista = [1, 2, 3, 4, 5]
elemento = random.choice(lista)  # Elige un elemento aleatorio
print(elemento)

datetime - Fecha y Hora

import datetime

hoy = datetime.datetime.now()
print(hoy)  # Fecha y hora actual

fecha = datetime.datetime(2026, 3, 31)
print(fecha)

os - Operaciones del Sistema

import os

# Obtener el directorio actual
print(os.getcwd())

# Listar archivos
print(os.listdir('.'))

# Crear carpeta
os.mkdir('nueva_carpeta')

Librerías Populares (terceros)

  • NumPy: Cálculos numéricos avanzados
  • Pandas: Análisis y manipulación de datos
  • Matplotlib: Gráficos y visualización
  • Django: Desarrollo web
  • TensorFlow/PyTorch: Inteligencia artificial
  • Pygame: Desarrollo de videojuegos

Importar Librerías

# Importar módulo completo
import math

# Importar función específica
from math import sqrt

# Importar todo (no recomendado)
from math import *

# Importar con alias
import numpy as np

Ver Funciones de una Librería

import math

print(dir(math))  # Lista todas las funciones disponibles
help(math.sqrt)   # Ayuda sobre una función específica

9. Automatización de Tareas

Python es muy popular para automatizar tareas repetitivas, lo que ahorra tiempo y esfuerzo. Una sola línea de código puede hacer lo que normalmente requeriría múltiples clics manuales.

Tareas Comunes de Automatización

  • Procesar archivos
  • Organizar datos
  • Enviar correos automáticos
  • Renombrar archivos en masa
  • Descargar contenido web
  • Realizar backups

Ejemplo: Simulación de Procesamiento de Archivos

for i in range(1, 6):
    print("Procesando archivo", i)
    print("- Leyendo datos...")
    print("- Analizando información...")
    print("- Guardando resultados...")
    print("✓ Archivo", i, "completado\n")

Ejemplo: Renombrar Archivos

import os

# Ejemplo conceptual
archivos = ["imagen1.jpg", "imagen2.jpg", "imagen3.jpg"]

for i, archivo in enumerate(archivos):
    nuevo_nombre = f"foto_{i+1:02d}.jpg"
    print(f"Renombrando {archivo} a {nuevo_nombre}")

Ejemplo: Lectura de Archivos

with open("datos.txt", "r") as archivo:
    contenido = archivo.read()
    print(contenido)

# Procesar línea por línea
with open("datos.txt", "r") as archivo:
    for linea in archivo:
        print(linea.strip())  # strip() elimina espacios en blanco

Ejemplo: Crear Archivos Masivamente

for mes in range(1, 13):
    nombre_archivo = f"reporte_{mes:02d}.txt"
    with open(nombre_archivo, "w") as archivo:
        archivo.write(f"Reporte del mes {mes}\n")
        archivo.write("Datos...\n")
    print(f"Creado: {nombre_archivo}")

Ventajas de la Automatización

  • Ahorro de tiempo: Tareas que toman horas se hacen en segundos
  • Menos errores: Eliminan errores humanos
  • Productividad: Te libera para otras tareas
  • Consistencia: Resultados siempre iguales
Ejemplo del Mundo Real

Una empresa necesitaba procesar 10,000 archivos de datos. Manualmente tomaría meses. Con Python, se automatizó y completó en minutos.
